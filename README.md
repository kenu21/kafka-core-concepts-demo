# kafka-core-concepts-demo

Educational project that demonstrates all core Apache Kafka concepts:
producers, consumers, partitions, ordering, delivery semantics,
offset management, consumer groups, rebalance, retention, replication,
Kafka Streams, Kafka Connect, monitoring and KRaft mode.

## Goals
- Understand Kafka as a distributed log (not a queue)
- Learn producer / consumer internals
- Demonstrate delivery guarantees
- Cover interview-level Kafka topics with working code
